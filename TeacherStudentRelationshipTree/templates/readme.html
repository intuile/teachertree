README

1.开发人员：李铮， 慕飞廷， 孙雨菁


2.安装环境：Python3.7+flask+前端渲染


3.项目背景：师生之间的沟通是搞好教学工作的重要条件之一，随着时代的发

展，有可能一个老师已经忘记了他所教导的学生，他的学生也可能会去做老师

所以建立一个师生图谱是费城必要的，也就是我们这个项目所希望达成的目的


4.项目目的：为了方便管理同校师生间信息，为了加强师生之间的联系，使相处

过的老师，学生显而易见，避免忘记，建立一个比较方便的图谱。


5.使用说明：首先进入登陆页面进行注册，然后登陆进去，在软件内部可以添加

也在使用该软件的老师或者学生，前提是要知道这个人的的账号，在

主界面显示和你相关的师承树。还有其他的修改信息，删除关系，查

找他人关系等操作。
